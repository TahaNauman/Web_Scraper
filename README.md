# Web Scraper with Requests & BeautifulSoup

## Description
This is a simple web scraper built with Python, Requests, BeautifulSoup, and Fake-UserAgent.  
It allows you to:
- Fetch a single webpage
- List all HTML tags found on the page
- Select tags to extract text from
- Save the extracted content to a JSON file

## Features
- Random User-Agent for polite scraping
- Skips empty content
- Saves output in UTF-8 for proper character handling
- Easily extensible for multi-page scraping

## Requirements
- Python 3.x
- requests
- beautifulsoup4
- fake-useragent

## Usage
```bash
python scraper.py
